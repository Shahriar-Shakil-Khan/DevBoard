Question-1: What are the different ways to select an element in the dom?
You can select elements using getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll, and more.

Question-2: What is the difference between innerHTML, innerText, and textContent?
innerHTML: Gets/sets HTML content inside an element.
innerText: Gets/sets visible text content, excluding hidden elements.
textContent: Gets/sets all text content, including hidden elements.

Question-3: What is event delegation in the DOM?
Event delegation allows you to attach a single event listener to a parent element to handle events for child elements.

Question-4: What is event bubbling in the DOM?
Event bubbling refers to the process where an event triggered on a child element propagates up to its parent elements.
